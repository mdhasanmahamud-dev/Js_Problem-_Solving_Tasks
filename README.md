# JavaScript Problem Solving

This repository contains solutions to **10 basic JavaScript problems** using **plain JavaScript**.  
Focus is on **loops, conditionals, functions, arrays, strings, and core logic**.  

---

## Problems
1. Reverse a String  
2. Count Vowels  
3. Check for Palindrome  
4. Find Maximum Number  
5. Remove Duplicates  
6. Sum of Numbers  
7. Find Even Numbers  
8. Capitalize First Letter  
9. Factorial  
10. PingPong Challenge  

---

## Technologies
- JavaScript (ES6)
- No frameworks or libraries
